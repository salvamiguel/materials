# AWS JAM y Cierre del Curso <FaAws />

La ultima sesion del curso de preparacion para la certificacion AWS Certified AI Practitioner (AIF-C01) esta dedicada a un evento practico de AWS JAM y al cierre formal del curso, incluyendo un repaso de los temas clave y recomendaciones para los siguientes pasos.

## 1. Que es AWS JAM

**AWS JAM** es una experiencia de aprendizaje practica y gamificada ofrecida por Amazon Web Services. A diferencia de los laboratorios tradicionales, donde se siguen instrucciones paso a paso, un JAM presenta desafios abiertos que los participantes deben resolver utilizando sus conocimientos de AWS en entornos reales de la nube.

El formato JAM esta disenado para simular situaciones del mundo real en las que un profesional de la nube debe diagnosticar problemas, disenar soluciones e implementarlas utilizando servicios de AWS. No se proporcionan instrucciones detalladas; en su lugar, se describe el problema y el resultado esperado, y cada equipo debe encontrar su propio camino hacia la solucion.

La plataforma de AWS JAM esta disponible en [jam.aws.com](https://jam.aws.com/v2/).

:::info Experiencia practica real
Los entornos de AWS JAM son cuentas reales de AWS provisionadas especificamente para el evento. Esto significa que los participantes trabajan con los servicios reales, no con simulaciones, lo que proporciona una experiencia mucho mas autentica y valiosa.
:::

## 2. Como funciona AWS JAM

### Dinamica de equipos

AWS JAM se desarrolla en equipos, lo que fomenta la colaboracion y el intercambio de conocimientos entre los participantes. Cada equipo trabaja de forma conjunta para resolver los desafios, combinando las fortalezas individuales de sus miembros. Esta dinamica refleja el trabajo real en equipos de ingenieria cloud, donde la colaboracion es fundamental.

### Desafios y puntuacion

Los desafios en un JAM estan organizados por nivel de dificultad y por dominio tematico. Cada desafio otorga una cantidad de puntos proporcional a su dificultad:

| Aspecto | Descripcion |
|---------|-------------|
| Formato | Desafios abiertos en entornos reales de AWS |
| Equipos | Los participantes trabajan en grupos colaborativos |
| Entorno | Cuentas de AWS reales provisionadas para el evento |
| Puntuacion | Puntos por desafio resuelto, con bonus por velocidad |
| Pistas | Sistema de pistas disponible, pero reduce los puntos obtenidos |
| Duracion | Tiempo limitado para resolver el maximo numero de desafios |

El sistema de puntuacion incentiva tanto la precision como la velocidad. Los equipos que resuelven desafios rapidamente obtienen puntos adicionales, y existe un sistema de pistas que los participantes pueden utilizar cuando estan atascados, aunque hacerlo reduce la puntuacion del desafio.

### Temas cubiertos en el JAM

En el contexto de este curso, los desafios del JAM estan centrados en los servicios de IA y machine learning de AWS que son relevantes para la certificacion AIF-C01. Los participantes pueden enfrentarse a escenarios que involucran:

- Configuracion y uso de Amazon Bedrock y modelos fundacionales
- Implementacion de soluciones con Amazon SageMaker
- Uso de servicios de IA predefinidos como Amazon Comprehend, Textract y Rekognition
- Aplicacion de buenas practicas de IA responsable
- Diseno de arquitecturas que integren servicios de IA con otros servicios de AWS

:::tip Aprovecha el JAM al maximo
El JAM es una oportunidad unica para poner a prueba tus conocimientos en un entorno seguro y sin consecuencias. No tengas miedo de experimentar y probar diferentes enfoques. Los errores durante un JAM son oportunidades de aprendizaje, no fallos.
:::

## 3. Aprendizajes clave del JAM

La experiencia del JAM refuerza varios aspectos que son dificiles de adquirir solo con estudio teorico:

**Resolucion de problemas bajo presion**: el tiempo limitado obliga a tomar decisiones rapidas y priorizar acciones, una habilidad crucial tanto para el examen como para el trabajo real con AWS.

**Conocimiento practico de la consola de AWS**: navegar por la consola, encontrar configuraciones y utilizar los servicios de forma efectiva es algo que solo se aprende con la practica. El JAM proporciona esa experiencia de forma concentrada.

**Comprension profunda de los servicios**: al enfrentarse a problemas reales, los participantes descubren matices de los servicios que no aparecen en la documentacion teorica. Este conocimiento practico es invaluable para el examen de certificacion.

**Trabajo en equipo**: la colaboracion con otros participantes expone a diferentes perspectivas y enfoques, ampliando la comprension de como se pueden utilizar los servicios de AWS.

## 4. Resumen del curso y conceptos clave

A lo largo de las 16 sesiones de este curso, se han cubierto todos los dominios del examen de certificacion AWS Certified AI Practitioner (AIF-C01):

- **Fundamentos de IA y ML**: conceptos basicos de inteligencia artificial, machine learning, deep learning y sus diferencias
- **IA generativa**: modelos fundacionales, LLMs, arquitecturas transformer, tecnicas de prompting y fine-tuning
- **Servicios de IA de AWS**: Amazon Bedrock, SageMaker, Comprehend, Textract, Rekognition, Polly, Transcribe y otros servicios gestionados
- **Bedrock Agents y MCP**: arquitectura de agentes autonomos, Knowledge Bases, Guardrails y el Model Context Protocol
- **IA responsable**: principios eticos, sesgo algoritmico, equidad, transparencia y gobernanza de modelos de IA
- **Seguridad y cumplimiento**: modelo de responsabilidad compartida, cifrado, control de acceso y proteccion de datos en servicios de IA

:::info Dominios del examen
El examen AIF-C01 evalua cinco dominios principales: Fundamentos de IA y ML, Fundamentos de IA generativa, Aplicaciones de modelos fundacionales, Directrices de IA responsable, y Seguridad y cumplimiento de soluciones de IA. Asegurate de sentirte comodo con todos ellos antes de presentarte al examen.
:::

## 5. Proximos pasos recomendados

Una vez finalizado el curso, estos son los pasos recomendados para completar tu preparacion y obtener la certificacion:

### Presentarse al examen

Si has completado las actividades de ETC y has obtenido tu voucher gratuito, el siguiente paso logico es programar tu examen de certificacion. Se recomienda no esperar demasiado tras finalizar el curso para no perder el impulso del estudio. Idealmente, programa el examen dentro de las dos semanas siguientes.

### Continuar practicando

Antes del examen, repasa los simulacros realizados durante el curso, prestando especial atencion a las preguntas que fallaste. Revisa tambien los laboratorios practicos para refrescar tu conocimiento de los servicios. Si necesitas practica adicional, AWS Skill Builder ofrece examenes de preparacion adicionales.

### Explorar certificaciones avanzadas

Una vez obtenida la certificacion AI Practitioner, puedes considerar avanzar hacia certificaciones de nivel superior:

| Certificacion | Nivel | Enfoque |
|---------------|-------|---------|
| AWS Certified Machine Learning - Specialty | Specialty | ML en profundidad con SageMaker |
| AWS Certified Solutions Architect - Associate | Associate | Arquitectura general de AWS |
| AWS Certified Data Engineer - Associate | Associate | Ingenieria de datos en AWS |

### Recursos adicionales

Para continuar tu preparacion y mantenerte actualizado en el ecosistema de IA de AWS, estos recursos son especialmente utiles:

- **AWS Skill Builder**: plataforma oficial de formacion con cursos, laboratorios y examenes de practica
- **AWS Documentation**: documentacion oficial de cada servicio, siempre actualizada
- **AWS re:Post**: foro de la comunidad AWS para resolver dudas tecnicas
- **AWS Blogs**: articulos sobre nuevas funcionalidades y casos de uso, especialmente el blog de Machine Learning
- **AWS Well-Architected Framework - Machine Learning Lens**: guia de buenas practicas para arquitecturas de ML en AWS
