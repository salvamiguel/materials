# Programa de Certificación AWS ETC <FaAws style={{ verticalAlign: 'middle' }} />

AWS ofrece un programa llamado **Emerging Talent Community (ETC)** que permite obtener certificaciones oficiales de forma totalmente gratuita. Este programa esta dirigido a estudiantes y profesionales que desean validar sus conocimientos en la nube de AWS sin coste alguno, siempre que completen una serie de actividades formativas dentro de la plataforma AWS Educate.

## 1. AWS Emerging Talent Community (ETC)

El programa AWS Emerging Talent Community es una iniciativa de Amazon Web Services orientada a fomentar el talento emergente en tecnologias cloud. A traves de este programa, los participantes pueden acumular puntos completando cursos y actividades en la plataforma AWS Educate, y canjear esos puntos por vouchers gratuitos para presentarse a examenes de certificacion oficial de AWS.

La propuesta de valor es clara: en lugar de pagar el coste habitual de un examen de certificacion (que puede rondar los 100-300 USD dependiendo del nivel), los participantes invierten tiempo en formarse y, a cambio, obtienen el acceso gratuito al examen.

:::info Vigencia del programa
El programa ETC esta sujeto a disponibilidad y puede tener fechas limite. Consulta la plataforma AWS Educate para verificar las condiciones actuales antes de comenzar.
:::

## 2. Pasos para obtener la certificacion gratuita

El proceso para conseguir tu certificacion a traves de ETC sigue cinco pasos bien definidos:

### Paso 1: Registrarse en AWS Educate

El primer paso es crear una cuenta en [awseducate.com](https://www.awseducate.com). El registro es gratuito y esta abierto a cualquier persona interesada en aprender sobre AWS. Una vez registrado, tendras acceso a un catalogo de cursos y recursos formativos.

### Paso 2: Completar el curso "Introduccion a la IA generativa"

Dentro del catalogo de AWS Educate, debes completar al menos un curso cualificante. El curso recomendado es **"Introduccion a la IA generativa"**, ya que es el mas corto de los cursos validos y esta directamente relacionado con la certificacion AWS AI Practitioner. Existen otros cursos validos, pero este ofrece la mejor relacion entre tiempo invertido y relevancia para el examen.

### Paso 3: Recibir la invitacion a ETC

Una vez completado el curso, AWS te invitara a unirte al programa Emerging Talent Community. Esta invitacion llega de forma automatica tras la finalizacion del curso cualificante. A partir de este momento, podras comenzar a acumular puntos dentro del ecosistema ETC.

### Paso 4: Completar actividades para ganar puntos

Dentro de ETC, cada actividad formativa otorga una cantidad determinada de puntos. El umbral necesario depende del nivel de certificacion al que aspires:

| Nivel de certificacion | Puntos necesarios |
|------------------------|-------------------|
| Foundational (ej. AI Practitioner, Cloud Practitioner) | 4500 puntos |
| Associate y superiores | 5200 puntos |

Las actividades incluyen cursos adicionales, laboratorios practicos, quizzes y otros recursos de aprendizaje disponibles en la plataforma.

### Paso 5: Completar el curso de AWS Skill Builder y el examen de preparacion

Finalmente, debes completar el curso oficial de preparacion en **AWS Skill Builder** correspondiente a la certificacion que deseas obtener (en nuestro caso, AWS Certified AI Practitioner AIF-C01) y realizar el examen de preparacion oficial. Esto no solo te otorgara puntos adicionales, sino que te preparara adecuadamente para el examen real.

:::tip Estrategia recomendada
Combina el estudio del material teorico de este curso con las actividades de ETC. De esta forma, acumularas puntos mientras te preparas para el examen, optimizando tu tiempo al maximo.
:::

## 3. Segundo simulacro de examen

Como parte de la preparacion, se realiza un segundo simulacro de examen que replica las condiciones del examen oficial de certificacion AWS Certified AI Practitioner (AIF-C01).

| Parametro | Valor |
|-----------|-------|
| Numero de preguntas | 65 |
| Tiempo disponible | 100 minutos |
| Puntuacion para aprobar | 46/65 (aproximadamente 70%) |

Este simulacro permite evaluar el progreso respecto al primer simulacro realizado anteriormente y detectar areas que todavia necesitan refuerzo antes del examen real. Es importante tomarse el simulacro con la misma seriedad que el examen oficial: sin consultar apuntes, respetando el limite de tiempo y en un entorno sin distracciones.

:::info Revision post-simulacro
Tras completar el simulacro, revisa cada pregunta fallada para entender por que la respuesta correcta es la que es. Las preguntas del examen real siguen patrones similares, por lo que esta revision es una de las actividades de estudio mas rentables.
:::

## 4. Laboratorio: Amazon Comprehend Custom Classification

La parte practica de esta sesion se centra en el servicio **Amazon Comprehend**, concretamente en su funcionalidad de clasificacion personalizada (Custom Classification).

Amazon Comprehend es un servicio de procesamiento de lenguaje natural (NLP) completamente gestionado que utiliza machine learning para extraer informacion y relaciones de textos. La clasificacion personalizada permite entrenar modelos que categorizan documentos segun las categorias especificas que defina el usuario, en lugar de limitarse a las categorias predefinidas del servicio.

En este laboratorio de AWS Skill Builder, los estudiantes trabajan directamente con Amazon Comprehend para:

- Configurar un proyecto de clasificacion personalizada
- Preparar y cargar datos de entrenamiento etiquetados
- Entrenar un modelo de clasificacion adaptado a un caso de uso especifico
- Evaluar los resultados del modelo y su precision

Este laboratorio refuerza los conceptos de **NLP** y **machine learning supervisado** que aparecen frecuentemente en el examen de certificacion, y proporciona experiencia practica con uno de los servicios de IA de AWS mas relevantes para la prueba.
